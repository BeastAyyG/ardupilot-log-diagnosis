# Neural Guard - React Dashboard

## Requirements
- Node.js 18+
- npm or yarn

## Installation
```bash
npm install
```

## Running the Development Server
```bash
npm run dev
```

## Building for Production
```bash
npm run build
```

## API Connection
The dashboard connects to the FastAPI backend at http://localhost:8000 by default.
You can configure the API URL in the environment variables.
